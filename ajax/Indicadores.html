Registre os dados de produtividade de sua empresa de forma a mantê-los organizados e disponíveis na web para consulta e envio. As principais funcionalidades do módulo são:


Download de template de gráficos de indicadores para suporte
Upload de indicadores prontos para o sistema, indexando-os com dados para posterior busca
Envio de link de visualização para qualquer destinatário com a finalidade de compartilhar a informação
Pesquisa detalhada atraés de qualquer informação utilizada para cadastrar os documentos
