Registre as atas de Análise Crítica pela Direção de sua empresa de forma a mantê-las organizadas e disponíveis na web para consulta e envio. As principais funcionalidades do módulo são:


Download de template de ata de reunião para suporte
Upload de atas prontas para o sistema, indexando-as com dados para posterior busca
Envio de link de visualização para qualquer destinatário com a finalidade de compartilhar a informação
Pesquisa detalhada através de qualquer informação utilizada para cadastrar os documentos
