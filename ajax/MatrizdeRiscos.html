Registre os riscos de sua empresa de forma a mantê-los organizados e disponíveis na web para consulta e envio. As principais funcionalidades do módulo são:


Download de template de matriz de risco para suporte
Cadastramento de riscos diretamente no sistema, indexando-os com dados para posterior busca
Envio de link de visualização para qualquer destinatário com a finalidade de compartilhar a informação
Pesquisa detalhada através de qualquer informação utilizada para cadastrar os documentos
Exportação da matriz de riscos cadastrada no sistema, por área ou geral
