Registre os Planos de Ação de sua empresa de forma a mantê-los organizados e disponíveis na web para consulta e envio. As principais funcionalidades do módulo são:


Download de template de Plano de Ação para suporte
Upload de Planos de Ação prontos para o sistema, indexando-os com dados para posterior busca
Envio de link de visualização para qualquer destinatário com a finalidade de compartilhar a informação
Vinculação de Planos criados com os registros que lhe deram origem
Pesquisa detalhada através de qualquer informação utilizada para cadastrar os documentos
